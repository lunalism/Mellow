# Mellow

**일상의 따뜻한 순간을 담는, 필름 감성 카메라 다이어리**

*A warm, faded film camera & mini-vlog diary for iOS*

</div>

---

Mellow(멜로우)는 잘 찍은 사진을 위한 앱이 아니라, **오늘을 남기기 위한 앱**입니다.
한낮의 볕, 카페의 BGM 같은 편안함 — 그 온도를 그대로 담을 수 있게,
따뜻하게 바랜 필름 색감과 미니멀한 화면, 그리고 짧은 영상 일기를 한곳에 모았습니다.

순백도 순흑도 쓰지 않습니다. 화면 어디에도 차가운 검정 대신 **들린 블랙(`#3B362E`)** 이 깔리고,
크림빛 아이보리(`#F4EBDD`)와 허니·테라코타가 은은하게 받쳐줍니다.

---

## 무엇이 다른가

구닥, Dazz, FIMO, VSCO, PICNIC… 좋은 카메라 앱은 많습니다.
하지만 **따뜻한 페이드 + 미니멀 + 영상까지 품는 다이어리**, 이 셋을 한 번에 채우는 앱은 없었습니다.
Mellow는 바로 그 빈자리에 섭니다.

- **따뜻한 페이드 룩** — 화려하지 않게, 기억의 색으로
- **미니멀한 카메라** — 찍는 일에만 집중하게
- **영상 일기** — 사진과 짧은 영상을 같은 다이어리에

## 핵심 원칙

- **완전 로컬** — 서버도, 계정도 없습니다. 당신의 기록은 당신의 기기에만 있습니다.
- **비파괴** — 필터를 입혀 저장하지 않습니다. *깨끗한 원본 + 필터 정보*를 따로 보관해, 언제든 다른 룩으로 다시 볼 수 있습니다.
- **WYSIWYG** — 프리뷰에서 본 색이 곧 저장된 색입니다. 프리뷰·썸네일·갤러리가 하나의 필터 엔진을 공유합니다.
- **구독 없음** — 핵심 카메라는 무료, 전체 기능은 **한 번의 평생 잠금 해제**로. 매달 빠져나가는 요금은 없습니다.

## 필터

따뜻한 단어로 이름 붙인 큐레이션 8종. 인기 앱의 필터를 베끼지 않고, 필름 룩의 결을 참고해 **오리지널 CIFilter 체인**으로 직접 빚었습니다.

`Sunday` *(기본)* · `Honey` · `Breeze` · `Hush` · `Olive` · `Linen` · `Ember` · `Velvet`

> 시즌 한정 필터는 더 시적인 한글 이름으로 — *'스쳐간 여름밤'* 처럼.

## 만든 방식

| | |
|---|---|
| **플랫폼** | iOS 17+ (네이티브) |
| **UI** | SwiftUI |
| **카메라 · 영상** | AVFoundation |
| **필터 파이프라인** | Core Image · Metal (`MTKView`) |
| **저장** | 로컬 파일 + JSON 메타데이터 (원본 + filterID) |
| **결제** | StoreKit (일회성 평생 잠금 해제) |

## 진행 상황

**Phase 1 — 카메라 코어** 가 활발히 구현 중입니다.

- [x] 카메라 구동 · 권한 · 라이브 프리뷰
- [x] Metal 필터 파이프라인 (라이브 필터 프리뷰)
- [x] 셔터 + 비파괴 저장 (깨끗한 원본 + filterID)
- [x] 인앱 필름 다이어리 갤러리 (비파괴 렌더 그리드)
- [x] 필름풍 셔터 피드백 (들린 블랙 블링크)
- [ ] 사진 상세뷰 · 삭제 · 사진앱 내보내기
- [ ] 필터 룩 심화 + 나머지 필터 · 종횡비 확장

**다음 단계:** 영상 일기(비파괴 영상 필터) → 평생 잠금 해제 → 출시 준비
